# Room shell kit (CC0)

Place CC0 furniture/environment GLB files here for the bedroom shell.

**Recommended sources:** Kenney Furniture Kit, KayKit, or similar low-poly CC0 packs.

**MVP note:** The shipped MVP uses procedural meshes authored in `data/rooms/bedroom.json` via `RoomBuilder.gd`. Replace shell props by importing GLBs and updating room data references when ready.

## Expected imports

- `floor_tile.glb`
- `wall_panel.glb`
- `bed_frame.glb`
- `desk.glb`
- `chair.glb`
- `bookshelf.glb`

Import settings: scale 1.0, generate collisions off (hotspots handle interaction).
