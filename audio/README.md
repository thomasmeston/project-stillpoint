# Audio assets (optional)

Drop OGG files here to enable ambient and SFX. The game runs without them.

| File | Trigger |
|------|---------|
| `ambient_bedroom.ogg` | Loop on scene start |
| `click.ogg` | Hotspot interaction |
| `door_unlock.ogg` | Door padlock solved |

Recommended: quiet room tone, soft UI click, low door chain rattle.
